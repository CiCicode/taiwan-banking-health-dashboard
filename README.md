# 銀行監理指標視覺化儀表板 | 台灣銀行業財務分析與資本管理活用專案
```
Core Value: 自動彙總台灣各大銀行數據，將繁複的監理統計轉化為直觀的動態儀表板，深度剖析資本管理、資產品質與獲利結構。
Tech Stack: JavaScript Web Scraper + Vercel Serverless Architecture + Dynamic Frontend
```

### 網站連結：https://car-dashboard-mu-three.vercel.app

## 功能
```
- **資本適足率**：趨勢圖 / 同業比較 / 分布圖 / 最新排行，支援 CAR/CET1/Tier1
- **資產品質**：逾放比率、備抵呆帳覆蓋率、放款覆蓋率趨勢與散佈圖
- **獲利能力**：ROA/ROE 趨勢與同業比較 + 杜邦分析
```

## 資料來源
```
金融監督管理委員會銀行局
```
